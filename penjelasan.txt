cara kerja : menggunakan link, pertama untuk mencari idsekolah yang nanti akan di gunakan ke link ke 2, lalu link ke 2 akan digunakan untuk mendapatkan idkelas yang nanti akan di gunakan untuk link ke 3, nanti link ke 3 akan mendapatkan idsiswa yang akan di gunakan untuk mengambil data data siswa tersebut

Penjelasan Scraping.py:
1. MAIN_URL = 'https://{sekolah}.sekolahan.id/' (digunakan untuk mendapatkan cookie yang akan di gunakan untuk login)
2. LOGIN_URL = 'https://{sekolah}.sekolahan.id/login/proses' (digunakan untuk login)
3. DATA_SISWA_URL = 'https://{sekolah}.sekolahan.id/datasiswa/edit/{idsiswa}' (digunakan untuk mengambil info tambahan untuk siswa)
4. DATA_SISWA_URL_PDF = 'https://{sekolah}.sekolahan.id/studentrecord/printdata/{idsiswa}' (digunakan untuk mengambil info siswa tapi pdf)
5. DATA_GURU_URL = 'https://{sekolah}.sekolahan.id/dataguru' (digunakn untuk menghitung berapa banyak halaman dan akan mengambil id guru)
6. INFO_GURU_URL = 'https://{sekolah}.sekolahan.id/dataguru/cetakprofil/{idguru}' (digunakan untuk mendapatkan info tentang guru, dengan id yang di dapatkan dari link DATA_GURU_URL dan di simpan dalam bentuk pdf)
7. ALL_DATA_GURU_URL 'https://{sekolah}.sekolahan.id/dataguru/cetakguru/'
8. DATA_TENDIK_URL = 'https://{sekolah}.sekolahan.id/datatendik' (mengambil url tentang datatendik, dengan contoh output seperti ini "https://smk-pertiwi-kuningan.sekolahan.id/datatendik/cetakprofil/268" yang nanti akan di download dengan file pdf)
9. ALL_DATA_TENDIK_URL = 'https://{sekolah}.sekolahan.id/datatendik/cetaktendik/
10. PROFIL_SEKOLAH_URL = 'https://{Sekolah}.sekolahan.id/profilsekolah' (digunakn untuk mengambil informasi sekolah)
